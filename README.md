# E-commerce UI

A fully functional e-commerce frontend with a shopping cart built with React.

## Features

- Product grid with 12 sample products
- Search, category filter, and sort (price / rating)
- Add to cart with quantity control
- Slide-out cart drawer
- Cart state managed with Context API + useReducer
- Badges for bestseller / top rated / new products
- Checkout button (UI only)

## Tech Stack

- React 18
- Vite
- React Context API
- useReducer for cart state

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
